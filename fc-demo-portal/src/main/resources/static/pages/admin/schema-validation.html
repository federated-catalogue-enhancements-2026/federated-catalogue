Ontologies

Controls whether uploaded ontologies extend the catalogue's recognised type vocabulary.

  • On: custom credential types defined via rdfs:subClassOf resolve to a role.
  • Off: only types built into the trust framework resolve; uploads using custom subclasses are rejected.
